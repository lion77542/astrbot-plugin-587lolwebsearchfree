## 📦 安装方式

### 方式一：通过 AstrBot 后台安装
1. 打开 AstrBot 管理后台
2. 进入「插件」页面
3. 点击「安装插件」
4. 上传本插件的 zip 包

### 方式二：手动安装
将 `main.py` 和 `metadata.yaml` 复制到 AstrBot 插件目录：
```
~/.astrbot/plugins/astrbot_plugin_search/
├── main.py
└── metadata.yaml
```
