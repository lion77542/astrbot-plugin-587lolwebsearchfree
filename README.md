## 🎯 核心功能

### 📱 客户端插件 (main.py)

| 命令 | 说明 | 示例 |
|------|------|------|
| `/搜 关键词` | 智能搜索中文网页内容 | `/搜 今日新闻` |
| `/搜图 关键词` | 搜索图片 | `/搜图 猫咪` |
| `/搜索建议 [关键词]` | 获取搜索技巧 | `/搜索建议 新闻` |
| `/搜索状态` | 查看插件状态 | `/搜索状态` |
| `/清空缓存` | 清空搜索缓存 | `/清空缓存` |
| `/帮助` | 显示使用指南 | `/帮助` |

### 🤖 LLM 自动工具调用

插件为 LLM 提供了三个自动工具：

1. **web_search** - 联网搜索网页内容（自动优化中文搜索）
2. **image_search** - 搜索网络图片
3. **crawl_page** - 爬取网页内容，提取正文

> 💡 LLM 会自动识别需要搜索的场景并调用相应工具，无需用户手动操作！

### 🖥️ 搜索服务端 (search_server.py)

| 端点 | 说明 |
|------|------|
| `GET /search?q=关键词&engine=all&num=5` | 网页搜索 |
| `GET /images?q=关键词&num=3` | 图片搜索 |
| `GET /crawl?url=网址&max_chars=3000` | 网页爬取 |
| `GET /health` | 健康检查 |
| `GET /stats` | 性能统计 |
| `POST /clear_cache` | 清空缓存 |

## 📦 安装指南

### 方法一：从 GitHub 安装（推荐）

1. 打开 AstrBot 管理后台
2. 进入「插件」页面
3. 点击右下角「+」按钮
4. 选择「从链接安装」
5. 输入仓库地址：

```
https://github.com/lion77542/astrbot-plugin-587lolwebsearchfree
```

### 方法二：手动安装

```bash
# 克隆仓库
git clone https://github.com/lion77542/astrbot-plugin-587lolwebsearchfree.git

# 复制到插件目录
cp -r astrbot-plugin-587lolwebsearchfree /path/to/astrbot/data/plugins/

# 重启 AstrBot
```

### 方法三：部署搜索服务端

```bash
# 安装依赖
pip install flask httpx

# 启动服务端
python search_server.py

# 服务端默认监听 0.0.0.0:11191
```

> **注意：** 搜索服务端需要 Python 3.8+，建议使用 `waitress` 作为生产环境服务器：
> ```bash
> pip install waitress
> python search_server.py
> ```

## ⚙️ 配置说明

### 客户端配置 (metadata.yaml)

```yaml
config:
  search_api: "http://chat.587.lol:11191"  # 搜索 API 地址
  cache_size: 20     # 缓存大小（条目数）
  cache_ttl: 300     # 缓存生存时间（秒）
  timeout: 12        # 请求超时时间（秒）
  max_retries: 2     # 失败重试次数
```

### 服务端配置 (search_server.py - Config 类)

```python
class Config:
    MAX_CACHE = 50           # 缓存条目数
    CACHE_TTL = 300          # 缓存生存时间（秒）
    CONNECT_TIMEOUT = 2      # 连接超时
    READ_TIMEOUT = 8         # 读取超时
    CONNECTION_POOL_SIZE = 15 # 连接池大小
```

## 🔧 故障排除

### 搜索无结果？
1. 使用 `/搜索建议 关键词` 获取优化建议
2. 尝试使用更简单的关键词
3. 使用 `/清空缓存` 清除旧缓存

### 搜索返回乱码？
- 系统已自动修复中文乱码问题
- 如果仍有问题，请检查网络连接

### 服务端无法启动？
1. 检查端口是否被占用：`lsof -i :11191`
2. 安装依赖：`pip install flask httpx`
3. 查看日志：`cat /tmp/search_server.log`

## 📝 技术架构

```
用户 → AstrBot → 客户端插件(main.py) → 搜索服务端(search_server.py) → 必应中国(cn.bing.com)
                                          ↓
                                     LRU 缓存 → 熔断器 → 结果去重 → 返回
```

### 客户端插件
- 基于 `aiohttp` 异步 HTTP 客户端
- 智能搜索词优化器
- 轻量级 LRU 缓存
- 自动重试和错误恢复

### 搜索服务端
- 基于 `Flask` + `httpx` 异步架构
- 多层优化：缓存、熔断、重试
- 搜索引擎：必应中国（cn.bing.com）
- 自动中文乱码修复
