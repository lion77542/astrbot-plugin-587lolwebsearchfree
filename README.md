## 📦 安装方式

1. 打开 AstrBot 管理后台
2. 进入「插件」页面
3. 点击右下角 **「+」** 按钮
4. 选择 **「从链接安装」**
5. 输入仓库地址：

```
https://github.com/lion77542/astrbot-plugin-587lolwebsearchfree
```

## 🚀 技术架构

### 插件端优化
- **连接池复用** - aiohttp TCPConnector，10连接池，5保持连接
- **LRU内存缓存** - 30条结果缓存，5分钟 TTL，0ms 响应
- **DNS预解析** - 启动时预解析域名IP，减少DNS查询
- **连接预热** - 启动时预热连接，首次请求更快
- **超时控制** - 连接3秒，读取8秒，超时即断
- **结果截断** - 标题50字，内容100字，减少传输

### 服务端优化
- **连接池复用** - httpx AsyncClient，30连接池
- **LRU内存缓存** - 50条结果缓存，5分钟 TTL
- **引擎熔断** - 单引擎连续失败5次自动熔断10分钟
- **并发控制** - Semaphore(3) 限制并发
- **UA轮换** - 10个真实浏览器 UA 随机切换
- **交叉验证** - 多引擎结果排序，共现结果优先
- **零磁盘写入** - 纯内存运行，不写临时文件
