# 🔍 AstrBot 联网搜索插件

让 AstrBot 机器人拥有联网搜索能力，支持文本搜索和图片搜索。

## 功能特性

- **文本搜索**：聚合 Bing、DuckDuckGo 多引擎结果
- **图片搜索**：搜索网络图片
- **极速响应**：10ms 缓存命中，600ms 首次搜索
- **零依赖**：无需 shell 权限，纯 HTTP 调用
- **全员可用**：群内所有成员都能使用，无需管理员权限

## 安装方式

### 方式一：直接复制
将 `main.py` 和 `metadata.yaml` 复制到 AstrBot 插件目录：
```
~/.astrbot/plugins/astrbot_plugin_search/
├── main.py
└── metadata.yaml
```

### 方式二：通过 AstrBot 后台安装
1. 打开 AstrBot 管理后台
2. 进入「插件」页面
3. 点击「安装插件」
4. 上传本插件的 zip 包

## 使用方法

在 QQ 群或私聊中发送以下命令：

| 命令 | 说明 | 示例 |
|------|------|------|
| `/搜 关键词` | 文本搜索 | `/搜 AI新闻` |
| `/搜图 关键词` | 图片搜索 | `/搜图 猫咪` |
| `/搜索状态` | 检查服务状态 | `/搜索状态` |

## 配置说明

插件默认连接远程搜索 API：
```
http://151.242.85.89:11191
```

如需修改 API 地址，编辑 `main.py` 中的 `SEARCH_API` 变量。

## 技术架构

```
用户发送 /搜 xxx
    ↓
AstrBot 插件接收命令
    ↓
调用远程搜索 API (HTTP)
    ↓
返回搜索结果
    ↓
发送给用户
```

## 依赖

- Python 3.10+
- aiohttp（AstrBot 自带）
