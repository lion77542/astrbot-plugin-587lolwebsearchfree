# AstrBot 联网搜索插件 - 587.lol 公益搜索（轻量优化版）

## 🎯 特性

- **专为低配服务器优化**：1核 256MB内存 2G存储环境完美运行
- **轻量级设计**：内存占用极低，约 1-2MB
- **智能缓存**：LRU 缓存机制，减少重复请求
- **自动重试**：失败自动重试，提高稳定性
- **DNS 预解析**：减少 DNS 查询延迟
- **资源清理**：自动管理连接池，防止资源泄漏
- **参数化请求**：防止 URL 注入，更安全

## 📦 安装

### 从 GitHub 安装

1. 打开 AstrBot 管理后台
2. 进入「插件」页面
3. 点击右下角「+」按钮
4. 选择「从链接安装」
5. 输入仓库地址：

```
https://github.com/lion77542/astrbot-plugin-587lolwebsearchfree
```

## 🚀 使用方法

### LLM 工具调用

插件提供了三个 LLM 工具，可以自动被 AI 调用：

1. **web_search** - 联网搜索网页内容
2. **image_search** - 搜索网络图片  
3. **crawl_page** - 爬取网页内容

### 命令使用

| 命令 | 说明 | 示例 |
|------|------|------|
| `/搜 关键词` | 文本搜索 | `/搜 AI新闻` |
| `/搜图 关键词` | 图片搜索 | `/搜图 猫咪` |
| `/搜索状态` | 查看插件状态 | `/搜索状态` |
| `/清空缓存` | 清空搜索缓存 | `/清空缓存` |

## ⚙️ 配置说明

插件已针对低配服务器优化，默认配置如下：

- **缓存大小**：20 条（约 1-2MB 内存）
- **缓存 TTL**：300 秒（5 分钟）
- **请求超时**：12 秒
- **重试次数**：2 次
- **连接池**：5 个总连接，每主机 3 个

如需调整，可在 `metadata.yaml` 中修改配置。

## 🏗️ 技术架构

### 内存优化
- **轻量级缓存**：SimpleCache 类，限制最大条目数
- **连接池复用**：aiohttp TCPConnector，减少连接开销
- **DNS 预解析**：避免 DNS 查询延迟
- **自动资源清理**：插件终止时自动释放资源

### 稳定性优化
- **自动重试机制**：失败请求自动重试
- **超时控制**：防止请求挂起
- **错误处理**：完善的异常捕获和错误提示
- **缓存机制**：减少重复请求，降低服务器负载
- **参数化请求**：防止 URL 注入攻击

### 兼容性
- **保留原功能**：完全兼容原仓库的所有功能
- **向后兼容**：保留了 `_fetch` 方法，确保兼容性
